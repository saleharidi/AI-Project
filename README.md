# Neon Snake Game

An iconic snake game.

### Technologies
* HTML
* CSS
* JavaScript

### How to use:

1. open the webpage on your browser.
2. Press enter and play.
3. Control the snake with you arrow keys.
4. Collet appeles and special fruits.
5. Avoid obstacles.

### Features

* standerd snake game with a score system.
* lightning fruit for extra boost of speed.
* a grape fruit for a slow effect and extra control.
* a shiled that acts as a revive.
* an obstacle system to make the game more intense.
